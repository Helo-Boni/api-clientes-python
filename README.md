# API de Clientes em Python

Projeto de backend desenvolvido com Flask, simulando um sistema de gerenciamento de clientes.

## Funcionalidades
- Listar clientes
- Adicionar clientes
- Estrutura de API REST

## Tecnologias
- Python
- Flask

## Como executar
pip install flask
python app.py

## Objetivo
Praticar desenvolvimento backend e criação de APIs.
